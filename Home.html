The Java Group

This Java User Group (JUG) is a community of users of the Java programming language specially form all over Kolkata.
Typically, members of a JUG support each other through:

1. Regular meetings (e.g. monthly)
2. Mailing lists
3. A collaborative website or wiki
4. Study groups for Java certification exams
Like other technology user groups, this JUG covers costs by meeting in educational or community spaces (e.g. university lecture halls) and/or asking local companies to sponsor individual meetings.

And now this time it is Kolkata. The much awaited group has been finally opened by some Java guys around Kolkata.

Motto:our motto is to spread Java, the core programming language among all. We are here with all the latest technologies of Java like J2SE 8, Project Avatar etc.
